# CDEK Internship Chatbot - RAG Agent на LangGraph

Контекстный RAG-агент для консультирования пользователей по правилам международной стажировки "CdekStart".

## 📋 Описание

Бот отвечает на вопросы о программе стажировки, используя базу знаний из 5 текстовых файлов. 

**Ключевые особенности:**
- Поддержание контекста диалога через `session_id`
- Задает уточняющие вопросы если запрос неоднозначен (например, не указана страна)
- Не выдумывает ответы — только строгий ответ по базе знаний
- Гибкое подключение LLM (OpenAI или локальная модель)

## 🚀 Быстрый старт

### Вариант 1: Docker Compose (рекомендуется)

```bash
docker-compose up --build
```

Сервис будет доступен на http://localhost:8000

### Вариант 2: Локальный запуск

```bash
# Установка зависимостей
pip install -r requirements.txt

# Запуск сервера
uvicorn app.main:app --host 0.0.0.0 --port 8000
```

## 📡 API

### POST /chat

Основной эндпоинт для общения с ботом.

**Request:**
```json
{
  "message": "Какая ставка стипендии в Германии?",
  "session_id": "optional-session-id"
}
```

**Response:**
```json
{
  "response": "Ставка стипендии в Германии составляет 1200 евро в месяц.",
  "session_id": "generated-or-provided-session-id",
  "sources": ["Правила для Германии"]
}
```

### GET /health

Проверка работоспособности сервиса.

## 🔧 Настройка LLM

### Использование OpenAI

1. Получите API ключ на https://platform.openai.com
2. Добавьте переменную окружения в `docker-compose.yml`:
   ```yaml
   environment:
     - OPENAI_API_KEY=sk-your-api-key-here
   ```
3. Перезапустите сервис

### Локальная модель (по умолчанию)

По умолчанию используется модель `TinyLlama/TinyLlama-1.1B-Chat-v1.0` через HuggingFace. Модель загружается автоматически при первом запуске.

**Требования:** 
- Минимум 4GB RAM для локальной модели
- Рекомендуется 8GB+ для комфортной работы

### Другие варианты

В файле `app/main.py` функция `get_llm()` поддерживает подключение:
- Ollama (локально)
- Anthropic Claude
- Любая модель через LangChain

## 🧪 Примеры запросов

### Простой вопрос
```json
{"message": "Когда дедлайн подачи документов?"}
```

### Вопрос требующий уточнения
```json
{"message": "Какая ставка стипендии?"}
```
*Бот спросит: "Уточните, пожалуйста, какая страна вас интересует: Германия или Франция?"*

### Вопрос с контекстом
```json
{"message": "А какой там налог?", "session_id": "previous-session-id"}
```
*Бот поймет контекст предыдущего диалога*

## ⚠️ Ограничения

- Локальная модель может работать медленно на слабых машинах
- Для production рекомендуется использовать облачные LLM (OpenAI GPT-4, etc.)
- База знаний ограничена 5 файлами в папке `data/`

## 🛠 Технологии

- **FastAPI** — веб-фреймворк
- **LangGraph** — оркестрация агента
- **LangChain** — работа с LLM
- **ChromaDB** — векторное хранилище
- **Sentence Transformers** — эмбеддинги
- **Docker** — контейнеризация
